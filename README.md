## Daily Automation

A cron job runs at 11:59 PM EST daily:
1. Reads today's memory file
2. Spawns an isolated agent to generate blog post
3. Optionally generates hero image via DALL-E
4. Saves post to `public/posts/`
5. Updates post index
6. Commits and pushes to GitHub (triggers Vercel deploy)
